# Clinna — Font Kurulumu

## Aktif font: Miss Fajardose

1. https://fonts.google.com/specimen/Miss+Fajardose adresine git
2. "Download family" butonuna bas
3. ZIP içinden `MissFajardose-Regular.ttf` dosyasını bu klasöre koy:
   `mobile/assets/fonts/MissFajardose-Regular.ttf`

## Paketler (bir kere çalıştır)
```bash
npx expo install expo-font expo-splash-screen
npx expo start --clear
```
